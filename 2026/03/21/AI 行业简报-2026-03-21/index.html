OpenAI 收购 Python 工具商 Astral 对抗 Anthropic

OpenAI 宣布收购 Python 工具开发商 Astral，后者是 uv 和 ruff 等热门开发者工具的创造者。此次收购旨在加强 OpenAI 在 AI 编码工具市场的竞争力，对抗 rival Anthropic。交易条款未披露，但 uv 每月下载量超过 1.26 亿次，ruff 比传统工具快 1000 倍。

AI 编码初创公司 Cursor 计划推出新模型

AI 编码初创公司 Cursor 计划推出新的大型语言模型，旨在与 Anthropic 和 OpenAI 竞争。Cursor 支持来自 OpenAI 和 Anthropic 的多种模型，并已将 ChatGPT 制造商列为投资者之一。此举标志着 AI 编码工具市场竞争进一步加剧。

Anthropic 与 OpenAI 谈论安全，但团队人手不足
